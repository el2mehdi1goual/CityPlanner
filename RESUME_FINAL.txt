🚀 DÉMARRAGE RAPIDE (5 MINUTES)
════════════════════════════════════════════════════════════════════════════════

Step 1: Créer l'environnement virtuel
────────────────────────────────────
Windows:
  python -m venv venv
  venv\Scripts\activate

Linux/macOS:
  python3 -m venv venv
  source venv/bin/activate


Step 2: Installer les dépendances
──────────────────────────────────
  pip install -r requirements.txt


Step 3: Créer la base de données MySQL
───────────────────────────────────────
Dans MySQL:
  CREATE DATABASE ai_urban_design CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;


Step 4: Appliquer les migrations
────────────────────────────────
  python manage.py makemigrations
  python manage.py migrate


Step 5: Créer un compte admin
────────────────────────────
  python manage.py createsuperuser
  (Entrez: username, email, password)


Step 6: Lancer le serveur
───────────────────────
  python manage.py runserver


Step 7: Accéder à l'application
──────────────────────────────
  http://127.0.0.1:8000

💡 EXEMPLES D'UTILISATION
════════════════════════════════════════════════════════════════════════════════

Créer un projet:
  1. S'inscrire: http://127.0.0.1:8000/register/
  2. Se connecter: http://127.0.0.1:8000/login/
  3. Cliquer: "Nouveau projet"
  4. Remplir: 
     - Titre: "Relogement Zone A"
     - Familles: 500
     - Taille moyenne: 5
     - Terrain: 75 hectares
     - Budget: Moyen
     - Priorité: Équilibré
  5. Soumettre

Générer une proposition:
  1. Aller sur: /projets/1/ (détails du projet)
  2. Cliquer: "Générer proposition"
  3. Voir les résultats (écoles, santé, marchés, espaces verts, routes)

Générer un prompt image:
  1. Cliquer: "Générer prompt image"
  2. Copier le prompt
  3. Utiliser avec: Midjourney, DALL-E, Stable Diffusion, etc.

🎯 ALGORITHME DE GÉNÉRATION
════════════════════════════════════════════════════════════════════════════════

Exemple: 500 familles, 5 personnes par famille, 75 hectares

Population = 500 × 5 = 2,500 habitants

Logements = 500 unités
Écoles = CEIL(2500/500) = 5
Centres de santé = CEIL(2500/1000) = 3
Marchés = CEIL(2500/800) = 4

Espaces verts:
  • Priorité "Espaces verts" → 75 × 15% = 11.25 ha
  • Priorité "Équilibré" → 75 × 10% = 7.5 ha
  • Autres priorités → 75 × 7% = 5.25 ha

Routes = 75 × 20% = 15 hectares

Reste pour habitations, services = Total - Routes - Espaces verts

Q: Comment modifier les paramètres de la base de données?
R: Éditer config/settings.py, section DATABASES

Q: Comment ajouter de nouveaux champs à un projet?
R: Éditer urban_design/models.py, puis makemigrations et migrate

Q: Comment créer une nouvelle page?
R: Ajouter une vue dans views.py, une route dans urls.py, un template
